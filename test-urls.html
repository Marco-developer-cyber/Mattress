Тест уникальных URL для товаров

Примеры URL:

Старый формат: #product-1

Новый формат: #product-1-orto-standart

Тестовый модал: #product-1-orto-standart-test

Тестовые URL для проверки:

#product-1-orto-standart - Детальная информация
#product-1-orto-standart-test - Тестовый модал

Как это работает:

  • Функция createProductSlug() преобразует русские названия в латиницу
  • URL создается в формате: product-{id}-{slug}
  • При нажатии "Подробнее" URL обновляется автоматически
  • При прямом переходе по URL товар открывается автоматически

Примеры преобразования названий:

  • "Орто Стандарт" → "orto-standart"
  • "Детский Комфорт" → "detskiy-komfort"
  • "Премиум Люкс" → "premium-lyuks"
